Muni Metro

Subway Status

Is the Muni Metro subway working right now?

That's the question this webapp answers.

 

How it works

A background task loads the image of the Muni Metro control screen, runs it through an image classifier, and reports the state back to this webapp.

This project is free and open source. Check it out over on Github.
